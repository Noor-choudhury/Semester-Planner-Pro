## 📖 Overview

**Semester Planner Pro** is a lightweight, fast, and fully responsive academic planner built with **Vanilla JavaScript**.

Designed specifically for students, it combines a **Google Calendar-inspired monthly planner**, **weekly class routine manager**, **smart reminders**, and **offline data storage** into a single-page web application.

No accounts. No database. No internet required after loading.

---

## ✨ Features

### 📅 Smart Monthly Calendar

* Beautiful Google Calendar-inspired interface
* Highlights today's date automatically
* Past dates marked with visual indicators
* Multi-line task support (up to 4 wrapped lines)
* Task completion toggle
* Quick edit & delete
* Detailed task modal
* Adjustable calendar zoom (50%–200%)

---

### 📚 Weekly Class Routine

Manage your entire semester schedule effortlessly.

* 📖 Multiple semesters
* 📅 Saturday–Thursday routine
* 🎨 Color-coded weekdays
* ➕ Add, Edit & Delete classes

* 🏫 Room numbers
* 📝 Course & section tracking
* ⏰ Start & End times

### 📊 Live Dashboard

Stay updated throughout the day.

* Today's classes

### 🔔 Smart Notifications

### 🌙 Customization

Personalize your experience.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/semester-planner-pro.git
```

Open the project:


## 📋 How to Use

### 1️⃣ Create a Semester

* Click the **Routine** icon
* Create a semester (e.g., *Fall 2026*)
* Save it

---

### 2️⃣ Add Your Weekly Routine

For each class, enter:

* Course Name
* Section
* Faculty
* Room
* Start Time
* End Time

---

### 3️⃣ Add Tasks

Create reminders for:

* Assignments
* Quizzes
* Exams
* Project deadlines
* Personal events

---

### 4️⃣ Stay Organized

The dashboard automatically shows:

* Today's classes
* Upcoming classes
* Today's tasks
* Tomorrow's tasks

---

### 5️⃣ Backup Your Planner

Use **Export** to download your planner as a JSON file.

Restore it anytime using **Import**.
